HTML5 e CSS

HTML5 é a quinta versão da linguagem de marcação HTML, usada para estruturar e apresentar conteúdo na web. CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a aparência e o formato de um documento escrito em HTML.

Java

Java é uma linguagem de programação de alto nível, orientada a objetos e amplamente utilizada para o desenvolvimento de aplicativos, desde aplicativos móveis e web até sistemas corporativos e grandes servidores.

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral, conhecida por sua sintaxe simples e legibilidade, o que facilita a aprendizagem e a manutenção do código.

Banco de Dados

Um banco de dados é um sistema organizado para armazenar, gerenciar e recuperar dados de forma eficiente e estruturada. Ele pode ser usado para armazenar informações de diferentes tipos, desde dados financeiros até perfis de usuários em um aplicativo.
